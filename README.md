# NightModeBot
A Telegram Bot to lock group in night to prevent night spam

## Setps To Use
```
- Put Variables Correctly.
- Add Bot to your group and make admin.
- That's all done
```

# Vars
```
CHAT_ID: Your Telegram Chat ID.
TOKEN: Your Telegram Bot Token.
```
